  • Overview:

    The Website team in the Apes of Wrath designs, builds, manages, and updates all pages on this domain. No prior knowledge is needed to join this category, although some knowledge in HTML, CSS, or Java Script would help greatly. Lessons will be assigned by Costi Khamis on online services such as Treehouse or W3 schools. This category is not as time consuming as other categories, so you are welcome to join other categories in addition to this one. Students in this category will learn basic website design, basic image design, and how to update and upload website files to a domain. All pages on this website are hand coded by the Website team.

    Skills:

    • HTML

    • CSS

    • JavaScript

    • Basic Image Editing

    • Basic Database Coding [Coming Soon]

    Managment:

    The current heads of the Website category are Nick Major and Costi Khamis.
